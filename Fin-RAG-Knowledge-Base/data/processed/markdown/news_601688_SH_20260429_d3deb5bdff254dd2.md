# 华泰证券：一季度净利润48亿元 同比增长31.79%

**来源**：证券时报网　**日期**：2026-04-29　**股票**：601688.SH

人民财讯4月29日电，华泰证券6016884月29日披露一季报，公司2026年一季度实现营业收入104.22亿元，同比增长41.48%（调整后）；净利润48亿元，同比增长31.79%（调整后）。因市场回暖，交投活跃，本期公司财富管理、机构服务、国际业务等主要业务线收入增加。
