# 【调研快报】晋控电力接待线上参与晋控电力0007672025年年度及2026年一季度业绩说明会的全体投资者调研

**来源**：东方财富Choice数据　**日期**：2026-05-12　**股票**：000767.SZ

晋控电力5月12日公告称，5月12日接待线上参与晋控电力0007672025年年度及2026年一季度业绩说明会的全体投资者调研。接待人员包括董事长、党委书记、总经理师 李军,独立董事、审计委员会主任委员 樊燕萍,总会计师 王晓华,董事会秘书 姚健。
