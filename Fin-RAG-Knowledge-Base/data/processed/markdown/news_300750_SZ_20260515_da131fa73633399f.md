# 宁德时代：截至2025年末，公司拥有在建产能321GWh，产能项目建设稳步推进

**来源**：每日经济新闻　**日期**：2026-05-15　**股票**：300750.SZ

宁德时代（300750.SZ）5月15日在投资者互动平台表示，截至2025年末，公司拥有在建产能321GWh，产能项目建设稳步推进。公司目前产能利用率较为饱和。
