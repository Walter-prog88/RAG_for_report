# 中信建投：第一季度净利润同比增长99.03%

**来源**：财联社　**日期**：2026-04-29　**股票**：601066.SH

中信建投601066.SH)公告称，2026年第一季度实现营业收入76.96亿元，同比增长62.26%；归属于母公司股东的净利润为36.67亿元，同比增长99.03%。业绩变动主要系公允价值变动损益、手续费及佣金净收入和利息净收入增加所致。
