# 5月18日龙虎榜净买入额前10个股

**来源**：南方财经网　**日期**：2026-05-18　**股票**：000988.SZ

南财智讯5月18日电，盘后数据显示，因当日异动登上龙虎榜的个股中，资金净买入最多的前三位分别是华工科技000988.SZ)、深科技(000021.SZ)与弘信电子(300657.SZ)。其中排名第一的华工科技净买额达16.99亿元，占总成交金额的17.37%。
