# 风华高科000636.SZ)：持股5%以上股东先进制造基金二期减持1.21%股份，持股降至5.58%

**来源**：界面新闻　**日期**：2026-05-13　**股票**：000636.SZ

2026年5月12日，广东风华高新科技股份有限公司（000636.SZ）发布公告称，公司持股5%以上股东先进制造产业投资基金二期（有限合伙）于2026年2月25日至5月11日期间，通过集中竞价交易方式减持公司股份14,001,400股，占公司总股本的1.21%。
