# 大秦铁路：4月大秦线货物运输量3116万吨 同比增长1.76%

**来源**：财联社　**日期**：2026-05-07　**股票**：601006.SH

大秦铁路601006.SH)公告称，2026年4月，公司核心经营资产大秦线完成货物运输量3116万吨，同比增长1.76%。2026年1—4月，大秦线累计完成货物运输量12802万吨，同比增长3.79%。
