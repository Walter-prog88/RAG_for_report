# 华泰证券：“因基金销售违规遭监管责令改正并暂停业务12个月”信息不实

**来源**：证券时报网　**日期**：2026-04-30　**股票**：601688.SH

人民财讯4月30日电，华泰证券601688午间公告，近日，公司关注到有市场媒体报道称华泰证券因基金销售违规遭监管责令改正并暂停业务12个月。经与各方核实，截至目前，公司未收到江苏证监局任何有关基金销售业务暂停12个月的行政监管措施。
