# 中联重科：具身智能机器人是公司培育的未来产业方向之一

**来源**：每日经济新闻　**日期**：2026-05-12　**股票**：000157.SZ

中联重科（000157.SZ）5月12日在投资者互动平台表示，具身智能机器人是公司培育的未来产业方向之一。机器人领域属于前沿科技、国家重点战略产业，目前行业整体研发及应用尚处于早期投资阶段，离规模性盈利仍需时间。敬请广大投资者谨慎决策，注意投资风险。
