# 1600亿存储巨头开启IPO辅导 这些A股公司或受益（附股）

**来源**：21世纪经济报道　**日期**：2026-05-19　**股票**：603156.SH

2025年4月，长江存储完成战略融资，养元饮品（603156.SH）公告旗下子公司泉泓投资出资16亿元投资长江存储。交易完成后，泉泓投资持有长江存储0.99%的股份。
