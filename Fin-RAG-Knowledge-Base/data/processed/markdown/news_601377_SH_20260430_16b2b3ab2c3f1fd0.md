# 兴业证券一季度归母净利润7.86亿元 同比增幅超五成

**来源**：央广财经　**日期**：2026-04-30　**股票**：601377.SH

4月29日，兴业证券（601377.SH）发布公告显示，公司2026年第一季度实现营业收入26.65亿元，较上年同期增长13.44%；归属于上市公司股东的净利润为7.86亿元，同比增幅达到52.17%。（央广财经）
