# 英科医疗接待138家机构调研 澄清高管跑路传闻

**来源**：深圳商报·读创　**日期**：2026-05-14　**股票**：300677.SZ

5月13日，英科医疗科技股份有限公司（300677.SZ）通过电话会议形式接待了宝盈基金、高毅资产、摩根士丹利等138家机构调研，公司管理层就市场关注的高管履职情况、订单价格、原材料成本及未来发展规划等核心问题作出详细回应，其中针对近期流传的高管“跑路”不实传闻进行明确澄清，强调全体高管均正常履职
