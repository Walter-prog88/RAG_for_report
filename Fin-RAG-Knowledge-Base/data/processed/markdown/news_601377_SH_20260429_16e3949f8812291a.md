# “债市一姐”刘郁拟加盟兴业证券，担任经济与金融研究院副院长

**来源**：南方财经网　**日期**：2026-04-29　**股票**：601377.SH

南方财经4月29日，21世纪经济报道记者获悉，兴业证券（601377.SH）拟聘任刘郁为经济与金融研究院副院长，分管固定收益研究中心。兴业证券方面表示，刘郁的加盟，将进一步助力研究院在资本市场研究领域持续发力。
