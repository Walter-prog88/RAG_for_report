# 又一艘万亿“券商航母”加速驶来！中金、东兴、信达“三合一”草案出炉，券商整合再掀热潮→

**来源**：金融投资网　**日期**：2026-05-19　**股票**：601995.SH

5月18日晚间，中金公司601995发布《换股吸收合并东兴证券股份有限公司、信达证券股份有限公司报告书(草案)》，显示其与东兴证券(601198)、信达证券(601059)换股吸收合并的万亿级“券商航母”已渐行渐近。
