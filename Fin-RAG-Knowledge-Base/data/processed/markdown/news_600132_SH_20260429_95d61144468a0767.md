# 重庆啤酒：第一季度净利润同比下降7.40%

**来源**：财联社　**日期**：2026-04-29　**股票**：600132.SH

重庆啤酒600132.SH)公告称，2026年第一季度实现营业收入43.50亿元，同比下降0.12%；归属于上市公司股东的净利润为4.38亿元，同比下降7.40%。小财注：公司Q1净利润4.38亿，2025年Q4净利润-0.1亿，据此计算，Q1净利润环比扭亏为盈。
