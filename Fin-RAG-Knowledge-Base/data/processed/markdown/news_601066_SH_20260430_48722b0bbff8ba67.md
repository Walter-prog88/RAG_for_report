# 中信建投保荐“踩红线”：尽职调查不充分被约谈

**来源**：财中社　**日期**：2026-04-30　**股票**：601066.SH

根据中国证监会北京监管局在2026年4月30日发布的公告，中信建投（601066）在证券发行保荐的个别项目中，对发行人的主要产品价格变动情况、部分主要客户和供应商、个别重要子公司等事项的尽职调查不充分,未审慎核查发行人申请文件和证券发行募集文件中有证券服务机构及其签字人员出具专业意见的内容
