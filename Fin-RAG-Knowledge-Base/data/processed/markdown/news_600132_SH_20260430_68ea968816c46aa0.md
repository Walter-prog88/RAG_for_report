# 重庆啤酒2026年首季营收超43亿元 高档产品持续增长多款新品加速落地

**来源**：证券时报网　**日期**：2026-04-30　**股票**：600132.SH

4月29日，重庆啤酒600132发布2026年第一季度报告，营业收入和净利润分别为43.5亿元和4.38亿元。 今年以来，重庆啤酒持续推进产品创新、产能布局与运营效率提升，延续了稳健发展态势，实现全年经营的良好开局，产品高端化稳步提升。
