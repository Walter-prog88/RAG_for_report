# 中信建投一季度归母净利润36.7亿元，同比增长99%

**来源**：财中社　**日期**：2026-04-30　**股票**：601066.SH

4月29日，中信建投601066公布2026年一季报，公司营业收入为77.0亿元，同比上升62.26%；归母净利润为36.7亿元，同比上升99.0%；扣非归母净利润为36.2亿元，同比上升97.4%；EPS(全面摊薄)为0.4728元。
