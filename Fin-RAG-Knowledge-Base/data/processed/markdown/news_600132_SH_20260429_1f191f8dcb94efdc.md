# 重庆啤酒：一季度净利润4.38亿元 同比下降7.4%

**来源**：证券时报网　**日期**：2026-04-29　**股票**：600132.SH

人民财讯4月29日电，重庆啤酒6001324月29日披露一季报，公司2026年一季度实现营业收入43.5亿元，同比下降0.12%；归母净利润4.38亿元，同比下降7.4%。报告期内，公司实现销量88.65万千升，比上年同期88.35万千升增长0.34%。
