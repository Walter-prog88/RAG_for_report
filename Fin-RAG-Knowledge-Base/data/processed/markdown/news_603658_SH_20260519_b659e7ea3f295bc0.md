# 安图生物：公司及全资子公司获多项医疗器械注册证

**来源**：财联社　**日期**：2026-05-19　**股票**：603658.SH

安图生物603658.SH)公告称，公司及全资子公司安图仪器近日收到国家药监局、河南省药监局颁发的医疗器械注册证，包括全自动化学发光免疫分析仪、EB病毒衣壳抗原IgM抗体检测试剂盒、肺炎支原体核酸及耐药突变位点检测试剂盒、白介素17A检测试剂盒。
