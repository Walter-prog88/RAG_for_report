# 长江存储控股完成IPO辅导备案 国产存储IDM双雄“奔赴”资本市场

**来源**：财联社　**日期**：2026-05-19　**股票**：603156.SH

去年4月，上市公司养元饮品（603156.SH）发布对外投资公告，称旗下子公司泉泓投资在2023年12月出资16亿投资长存集团，获得0.99%股权。以此估算，长存集团彼时的估值已超1600亿元。
