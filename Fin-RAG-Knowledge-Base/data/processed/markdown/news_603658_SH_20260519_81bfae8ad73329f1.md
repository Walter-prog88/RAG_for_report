# 安图生物及子公司斩获多项医疗器械注册证

**来源**：财中社　**日期**：2026-05-19　**股票**：603658.SH

5月19日，安图生物（603658）发布公告，公司及子公司安图仪器于近日收到国家药品监督管理局、河南省药品监督管理局颁发的医疗器械注册证，涉及多个产品，包括全自动化学发光免疫分析仪和EB病毒衣壳抗原IgM抗体检测试剂盒等，注册证有效期均为5年。
