# 挖掘机集体涨价！三一、徐工、柳工带头 内外需“双旺”支撑行业景气度

**来源**：华夏时报网　**日期**：2026-05-15　**股票**：600031.SH

5月以来，受原材料涨价、行业高景气双重驱动，三一重工（600031.SH）、徐工机械（000425.SZ）、柳工（000528.SZ）、山推股份（000680.SZ）等工程机械龙头相继发布挖掘机涨价通知，涨价幅度为3%—5%，将在5月中旬至6月初陆续落地。
