# 爱旭股份：会参加6月的上海国际太阳能光伏展，并将展出最新全系列ABC光伏组件产品及解决方案

**来源**：每日经济新闻　**日期**：2026-05-13　**股票**：600732.SH

爱旭股份（600732.SH）5月13日在投资者互动平台表示，公司会参加6月的上海国际太阳能光伏展，并将展出最新全系列ABC光伏组件产品及解决方案。
