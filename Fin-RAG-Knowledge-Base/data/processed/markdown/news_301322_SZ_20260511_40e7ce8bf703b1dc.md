# 绿通科技：137万股限制性股票5月13日上市流通

**来源**：财中社　**日期**：2026-05-11　**股票**：301322.SZ

5月11日，绿通科技（301322）发布公告，披露公司关于2025年限制性股票激励计划首次授予部分第一个归属期归属结果暨股份上市的情况，此次归属股票人数为100人，归属股票数量为137万股，占公司总股本的0.97%，上市流通日定于2026年5月13日，且不设限售期，授予价格为11.11
