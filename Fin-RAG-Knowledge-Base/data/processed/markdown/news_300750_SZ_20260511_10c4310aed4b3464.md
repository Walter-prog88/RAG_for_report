# 宁德时代：控股股东向上海交通大学捐赠500万股股份完成过户

**来源**：财联社　**日期**：2026-05-11　**股票**：300750.SZ

宁德时代300750.SZ)公告称，公司控股股东厦门瑞庭向上海交通大学教育发展基金会捐赠的500万股A股无限售流通股已完成过户。捐赠股份产生的全部收益及所得资金将注入“曾毓群教育发展基金”，用于支持上海交通大学教育事业发展。
