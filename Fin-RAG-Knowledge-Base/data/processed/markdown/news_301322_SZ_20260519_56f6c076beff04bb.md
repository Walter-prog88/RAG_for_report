# 净利连降后转亏，绿通科技拟1.2亿元控股诚瑞科技，标的已获A轮融资

**来源**：深圳商报·读创　**日期**：2026-05-19　**股票**：301322.SZ

5月19日盘后，绿通科技301322发布公告称，公司拟筹划以现金方式收购苏州市诚瑞科技有限公司（下称“诚瑞科技”）不低于51%的股权，本次交易完成后公司预计将实现对诚瑞科技的控股，预计交易对价不超过1.2亿元。 本次交易不构成关联交易。
