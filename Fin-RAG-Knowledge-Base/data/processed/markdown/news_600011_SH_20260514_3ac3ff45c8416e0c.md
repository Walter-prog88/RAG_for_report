# 华能国际间接控股子公司5.43亿元项目环评获原则同意

**来源**：每日经济新闻　**日期**：2026-05-14　**股票**：600011.SH

“A股绿色报告”项目监控到的数据显示，华能国际（SH600011）间接控股子公司华能津综（丹寨）新能源有限公司丹寨县南皋风电场项目环评审批获原则同意。该项目总投资额达5.43亿元。该审批信息于2026年4月24日被相关监管机构披露。
