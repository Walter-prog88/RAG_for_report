# 重庆啤酒：Q1归母净利润为4.38亿元，同比下降7.4%

**来源**：财中社　**日期**：2026-04-29　**股票**：600132.SH

4月29日，重庆啤酒600132公布2026年一季报，公司营业收入为43.5亿元，同比下降0.1%；归母净利润为4.38亿元，同比下降7.4%；扣非归母净利润为4.34亿元，同比下降7.1%；经营现金流净额为16.5亿元，同比增长22.25%；EPS(全面摊薄)为0.90元。
