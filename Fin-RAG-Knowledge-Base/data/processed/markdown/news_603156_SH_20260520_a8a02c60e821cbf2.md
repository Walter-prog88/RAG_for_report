# 乳蛋白炙手可热，植物蛋白饮品何时走出“降温区”？

**来源**：第一财经　**日期**：2026-05-20　**股票**：603156.SH

养元饮品（603156.SH）和承德露露（000848.SZ）2025年业绩双双下滑，同期国际燕麦奶巨头Oatly的收入结构也有所转变，餐饮业务收入占比在下降。 业内认为，相比于乳蛋白市场需求不断增长，植物蛋白赛道受制于创新不足，仍面临挑战。 “植物奶整体卖得都不算好。”
