# 北汽蓝谷：6.64亿股限售股将于5月25日上市流通

**来源**：财中社　**日期**：2026-05-18　**股票**：600733.SH

5月18日，北汽蓝谷（600733）发布公告，2023年定增中，控股股东关联方北汽股份、渤海汽车认购的约6.64亿股锁定期36个月，将于5月25日上市流通。公司总股本约63.67亿股，不存在控股股东及关联方资金占用情况。
