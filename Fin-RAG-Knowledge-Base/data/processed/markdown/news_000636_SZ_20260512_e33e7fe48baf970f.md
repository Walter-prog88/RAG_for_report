# 风华高科：先进制造基金二期减持1400万股 占总股本1.21%

**来源**：财中社　**日期**：2026-05-12　**股票**：000636.SZ

5月12日，风华高科（000636）发布公告，股东国投招商投资管理有限公司－先进制造产业投资基金二期于2026年2月25日至2026年5月11日通过集中竞价交易减持1400万股，占公司总股本的1.21%。
