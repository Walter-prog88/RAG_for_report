# 世嘉科技股价异动 公司称经营状况及内外部环境未现重大变化

**来源**：央广财经　**日期**：2026-05-17　**股票**：002796.SZ

5月17日，世嘉科技（002796.SZ）发布公告称，公司股票交易价格在连续三个交易日内收盘价格跌幅偏离值累计达20.24%，构成股票交易异常波动。公告显示，经核查，近期公司经营情况正常，内外部经营环境未发生重大变化。
