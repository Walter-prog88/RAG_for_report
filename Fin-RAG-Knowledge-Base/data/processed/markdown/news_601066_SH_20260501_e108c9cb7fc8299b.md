# 中信建投遭北京证监局约谈，因对保荐发行人尽职调查不充分

**来源**：南方都市报　**日期**：2026-05-01　**股票**：601066.SH

4月30日，北京证监局披露了关于对中信建投（601066.SH）采取监管谈话行政监管措施的决定。中信建投的两名工作人员罗贵均、包桉泰则均被采取了出具警示函的行政监管措施。
