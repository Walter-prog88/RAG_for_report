# 北汽蓝谷：近6.64亿股限售股将于5月25日解禁流通

**来源**：央广财经　**日期**：2026-05-19　**股票**：600733.SH

5月18日，北汽蓝谷（600733.SH）发布公告，披露其2023年向特定对象发行股票中，由控股股东关联方北汽股份及渤海汽车（600960.SH）认购的663,815,871股股份，因36个月锁定期即将届满，计划于2026年5月25日上市流通。
