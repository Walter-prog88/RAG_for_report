# 华泰证券某营业部员工郭秀因借用他人账户炒股被警示

**来源**：财中社　**日期**：2026-05-14　**股票**：601688.SH

根据天津证监局在2026年5月14日发布的公告，郭秀因在华泰证券（601688）天津勤俭道证券营业部从业期间，因借用他人证券账户买卖股票被出具警示函。
