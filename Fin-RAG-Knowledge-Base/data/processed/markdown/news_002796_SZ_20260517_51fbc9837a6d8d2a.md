# 世嘉科技：近期经营情况正常 内外部经营环境未发生重大变化

**来源**：财联社　**日期**：2026-05-17　**股票**：002796.SZ

世嘉科技002796公告称，公司股票连续三个交易日(5月13日至15日)收盘价格跌幅偏离值累计达20.24%，属于股票交易异常波动。经核查，公司前期披露信息无需更正补充，近期经营情况正常，内外部经营环境未发生重大变化。公司未发现公共传媒报道可能对股价产生较大影响的未公开信息。
