# 渤海租赁：计划于未来两年内交付的新飞机中已有约98%落实了租约

**来源**：每日经济新闻　**日期**：2026-05-13　**股票**：000415.SZ

渤海租赁（000415.SZ）5月13日在投资者互动平台表示，截至2025年末，公司机队整体出租率约100%，计划于未来两年内交付的新飞机中已有约98%落实了租约，计划于2028年内交付的新飞机中已有约74%落实了租约，平均剩余租约7年。
