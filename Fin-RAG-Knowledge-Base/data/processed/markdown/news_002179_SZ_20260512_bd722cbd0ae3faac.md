# 上证早知道｜688235，一季度扭亏为盈；002179，机构大买超3亿元；688287，终止上市

**来源**：上海证券报　**日期**：2026-05-12　**股票**：002179.SZ

资金观潮 ○机构席位净买入中航光电 5月11日交易公开信息显示，机构席位净买入中航光电（002179）3.13亿元，占总成交的7.07%。 东莞证券认为，中航光电是专业为航空及防务和高端制造提供互连解决方案的高科技企业，在多种类通信设备市场份额中保持领先地位。
