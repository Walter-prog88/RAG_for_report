# 血液制品业绩“大考”：多家龙头净利下滑 上海莱士“逆袭”

**来源**：21世纪经济报道　**日期**：2026-04-29　**股票**：002007.SZ

另一头部企业华兰生物（002007）4月26日公布2026年一季报，公司营业收入为7.62亿元，同比下降12.2%；归母净利润为2.3亿元，同比下降26.4%；扣非归母净利润为2.27亿元，同比下降24.3%；经营现金流净额为-639万元，同比下降145.1%。
