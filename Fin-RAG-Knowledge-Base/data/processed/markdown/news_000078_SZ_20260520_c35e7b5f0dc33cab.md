# ST海王连续3个交易日收盘价格跌幅偏离值累计超12%

**来源**：财中社　**日期**：2026-05-20　**股票**：000078.SZ

5月20日，ST海王（000078）发布公告，公司股票在2026年5月18日至20日连续3个交易日，收盘价格跌幅偏离值累计超过12%。 2026年一季度，ST海王实现收入59.45亿元，归母净利润1449万元。
