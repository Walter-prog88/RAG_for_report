# 比亚迪：公司在重卡领域拥有牵引车Q3、搅拌车T25、搅拌车T31、自卸车T25等多款产品

**来源**：每日经济新闻　**日期**：2026-05-07　**股票**：002594.SZ

比亚迪（002594.SZ）5月7日在投资者互动平台表示，公司多年深耕新能源商用车领域，其中在重卡领域拥有牵引车Q3、搅拌车T25、搅拌车T31、自卸车T25、自卸车T31、环卫底盘T18等多款产品。
