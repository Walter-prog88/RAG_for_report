# 16.99亿资金抢筹华工科技，机构狂买春光集团丨龙虎榜

**来源**：21世纪经济报道　**日期**：2026-05-18　**股票**：000988.SZ

盘后龙虎榜数据显示，共有63只个股因当日异动登上龙虎榜，资金净流入最多的是华工科技（000988.SZ），达16.99亿元。 据21投资通智能监测，39只个股的龙虎榜出现了机构的身影，北向资金参与龙虎榜中的个股共23只。
