# 中信建投：一季度净利润36.67亿元 同比增长99.03%

**来源**：证券时报网　**日期**：2026-04-29　**股票**：601066.SH

人民财讯4月29日电，中信建投6010664月29日披露一季报，公司2026年一季度实现营业收入76.96亿元，同比增长62.26%（调整后）；归母净利润36.67亿元，同比增长99.03%（调整后）。
