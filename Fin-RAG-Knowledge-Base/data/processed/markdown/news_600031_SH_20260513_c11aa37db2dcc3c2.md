# 挖掘机龙头联手提价 4月内销增逾三成：行业内卷有望反转？

**来源**：澎湃新闻　**日期**：2026-05-13　**股票**：600031.SH

5月以来，三一重工（600031）、徐工机械（000425）、柳工（000528）相继发布涨价通知。其中，三一重工宣布自5月15日起对挖掘机价格上调5%；柳工自5月20日起对挖掘机价格上调5%；徐工机械自6月1日起对不同机型上调3%至5%。
