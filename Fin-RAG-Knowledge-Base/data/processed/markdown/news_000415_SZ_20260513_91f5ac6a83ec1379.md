# 渤海租赁：公司境外负债主要以美元计价，美元降息有利于降低公司新增融资成本

**来源**：每日经济新闻　**日期**：2026-05-13　**股票**：000415.SZ

渤海租赁（000415.SZ）5月13日在投资者互动平台表示，1、截至2025年末，公司整体有息负债为1799.70亿元，较去年同期下降351.58亿元；资产负债率为81.26%，较2024年末下降1.71个百分点；平均融资成本为4.47%，较上年同期下降16个基点。
