# 跨界并购！金利华电、绿通科技一字涨停

**来源**：大河财立方　**日期**：2026-05-20　**股票**：301322.SZ

5月19日晚间，绿通科技（301322.SZ）公告称，拟以现金方式收购苏州市诚瑞科技有限公司（以下简称诚瑞科技）不低于51%股权，从而实现控股诚瑞科技之目的，本次交易预计交易对价不超过1.2亿元。
