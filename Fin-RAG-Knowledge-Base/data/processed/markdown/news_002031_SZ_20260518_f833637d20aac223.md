# 5月18日龙虎榜净卖出额前10个股

**来源**：南方财经网　**日期**：2026-05-18　**股票**：002031.SZ

南财智讯5月18日电，盘后数据显示，因当日异动登上龙虎榜的个股中，龙虎榜中资金净流出最多的前三位分别是优迅股份(688807.SH)、巨轮智能002031.SZ)与粤传媒(002181.SZ)。其中排名第一的优迅股份被净卖出4.93亿元，占总成交金额的22.5%。
