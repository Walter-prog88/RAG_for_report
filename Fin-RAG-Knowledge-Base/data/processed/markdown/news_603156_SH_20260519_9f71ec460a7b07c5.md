# 长江存储母公司启动上市辅导，估值超1600亿元

**来源**：南方都市报　**日期**：2026-05-19　**股票**：603156.SH

“六个核桃”生产商养元饮品（603156.SH）当时披露，旗下子公司泉泓投资已向长存集团投资16亿元。交易完成后，泉泓投资持有长存集团0.99%的股份。若按照16亿元持股0.99%股权进行计算，长存集团的市场估值约为1616亿元。
