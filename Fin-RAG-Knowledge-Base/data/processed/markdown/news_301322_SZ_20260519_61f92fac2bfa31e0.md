# 绿通科技拟1.2亿元收购诚瑞科技不低于51%股权

**来源**：证券时报网　**日期**：2026-05-19　**股票**：301322.SZ

绿通科技301322.SZ)公告称，公司拟以现金方式收购苏州市诚瑞科技有限公司（以下简称“诚瑞科技”）不低于51%股权，预计交易对价不超过1.2亿元。
