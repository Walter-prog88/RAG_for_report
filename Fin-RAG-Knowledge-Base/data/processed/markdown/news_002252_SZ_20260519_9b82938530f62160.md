# 上海莱士：2025年全年采浆量超过2000吨，同比增长超过8%，高于行业平均增幅

**来源**：每日经济新闻　**日期**：2026-05-19　**股票**：002252.SZ

上海莱士（002252.SZ）5月19日在投资者互动平台表示，2025年全年采浆量超过2000吨，同比增长超过8%，高于行业平均增幅，吨浆成本受原料、运营、浆站建设等多重因素综合影响，相关经营数据请以年报披露为准。
