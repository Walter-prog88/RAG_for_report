# 渤海租赁：公司以经营租赁为主的方式向全球范围内的航空公司及其他客户提供飞机中长期租赁服务

**来源**：每日经济新闻　**日期**：2026-05-13　**股票**：000415.SZ

渤海租赁（000415.SZ）5月13日在投资者互动平台表示，公司主营飞机租赁业务，以经营租赁为主的方式向全球范围内的航空公司及其他客户提供飞机中长期租赁服务。
