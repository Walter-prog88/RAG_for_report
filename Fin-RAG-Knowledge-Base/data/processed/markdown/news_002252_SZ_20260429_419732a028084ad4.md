# 血液制品业绩“大考”：多家龙头净利下滑 上海莱士“逆袭”

**来源**：21世纪经济报道　**日期**：2026-04-29　**股票**：002252.SZ

4月28日晚间，上海莱士（002252.SZ）披露了2026年一季度“成绩单”。报告期内，公司实现营业收入15.09亿元，归母净利润3.16亿元，归母扣非净利润3.20亿元，同比仍下滑，但三项指标环比数据可观，均实现20%以上的增长，其中归母净利润环比增幅达197.19%。
