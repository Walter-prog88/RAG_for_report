# 中信证券：公司已建立财务共享中心

**来源**：每日经济新闻　**日期**：2026-05-06　**股票**：600030.SH

中信证券（600030.SH）5月6日在投资者互动平台表示，公司已建立财务共享中心，以提升财务管理效能，支持公司业务发展，满足公司经营管理的需要。公司将持续探索人工智能、大数据等前沿技术在经营管理中的应用，加强数据治理和运用，持续推动公司数字化转型向全业务、全流程和全模式发展。
