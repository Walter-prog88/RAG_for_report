# “六个核桃”疲软 红牛托底：养元饮品要靠长江存储翻身？

**来源**：财中社　**日期**：2026-05-20　**股票**：603156.SH

养元饮品（603156）5月20日开盘一度大涨超 9%，之后有所回落，最终收报49.05元/股，涨幅1.43%。 此次早盘冲高，主要受国产存储龙头长江存储递交上市辅导备案、上市进程迎来实质进展的带动。
