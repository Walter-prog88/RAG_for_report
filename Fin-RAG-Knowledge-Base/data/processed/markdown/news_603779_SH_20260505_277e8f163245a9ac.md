# 603779，控制权拟变更！明天停牌

**来源**：e公司　**日期**：2026-05-05　**股票**：603779.SH

威龙股份（603779）控制权拟发生变更。 5月5日，威龙股份公告，公司于4月30日收到控股股东星河息壤（浙江）数智科技有限公司（以下简称“星河息壤”）通知，其正在筹划公司股份协议转让事宜，该事项可能导致公司控制权发生变更。
