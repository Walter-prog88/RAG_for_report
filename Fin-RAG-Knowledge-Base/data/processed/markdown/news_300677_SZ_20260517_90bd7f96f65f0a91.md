# 300677，火了！迎超130家机构调研！公司最新澄清

**来源**：证券时报网　**日期**：2026-05-17　**股票**：300677.SZ

“手套茅”澄清传闻 英科医疗300677上周接受138家机构调研，包括宝盈基金、北京润晖资产、财通资管、大成基金、摩根基金、兴业证券、中信证券等知名机构。英科医疗是一家综合型医疗护理产品供应商，因一次性手套市占率领先，被市场称为“手套茅”。
