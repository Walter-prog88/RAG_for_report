# 费城半导体指数 × 标普500看涨情绪 — 经济危机预警研究

> 研究目的：通过分析 SOX 相对强度与 SPX 看涨情绪的历史规律，判断当前是否处于经济危机前夕。

---

## 一、研究背景与核心假设

### 为什么看半导体？

半导体是现代经济的「神经元」——芯片需求先于终端消费产品需求变化，企业资本开支计划的收缩最先体现在半导体订单上。因此，费城半导体指数（Philadelphia Semiconductor Index, SOX）在历史上多次**领先标普500（SPX）6-12个月见顶或见底**。

### 两个历史案例

| 危机 | SOX 见顶 | SPX 见顶 | SOX 领先 | SOX 最大跌幅 |
|------|----------|----------|----------|------------|
| 科网泡沫（2000） | 2000-03-10 | 2000-03-24 | ~2周 | -82% |
| 次贷危机（2008） | 2007-10-05 | 2007-10-09 | ~4天（同步） | -60% |

> **注**：2000年泡沫中，SOX从1999年开始大幅领涨SPX，形成明显的半导体估值泡沫，这是最重要的先行信号；2008年次贷危机属于信用驱动型，SOX与SPX基本同步，无明显领先。

### 核心假设

```
高危组合 = SOX 相对 SPX 大幅领涨（半导体泡沫）
         × VIX 处于极低位（市场过度乐观，缺乏对冲）
         × SOX 远超长期均线（价格透支基本面）
```

当三者同时成立时，历史上对应的均是后续的系统性回调。

## 三、指标计算方法

### 3.1 基础价格指标

**归一化价格**（用于长期走势对比）
```
归一化价格 = 当日收盘价 / 起始日收盘价 × 100
```
将 SOX 和 SPX 统一换算为以1994年为基准的指数值，消除绝对价位差异，便于比较长期涨幅。

---

**SOX/SPX 相对强度比率**
```
比率 = SOX 归一化价格 / SPX 归一化价格
```
该比率升高 → 半导体领涨大盘；比率下降 → 半导体开始跑输。  
历史规律：比率在危机前往往达到局部峰值，然后率先反转。

---

### 3.2 波动率指标

**年化实现波动率**
```
对数日收益率 = ln(今日收盘 / 昨日收盘)
滚动年化波动率 = 标准差(过去N日对数收益率) × √252
```
使用两个窗口：
- **短期（21日 ≈ 1个月）**：捕捉近期波动是否突然放大
- **中期（63日 ≈ 3个月）**：代表当前的波动率「基准水平」

**波动率体制比率**
```
体制比率 = 短期波动率(21日) / 中期波动率(63日)
```
- 比率 ≈ 1.0：波动率稳定，市场平静
- 比率 > 1.5：短期波动突然高于中期基准，意味着危机已经开始扩散（**确认信号**，非领先信号）

---

### 3.3 情绪指标

**SOX 相对 SPX 6个月超额收益**
```
超额收益 = SOX 过去126个交易日涨幅 - SPX 过去126个交易日涨幅
```
持续为正且超过 +20% → 半导体估值泡沫化；转为负值 → 领先指标翻转，危机风险上升。

---

**VIX 低位指数（看涨情绪代理）**
```
VIX 低位指数 = 1 - VIX在过去2年中的百分位排名
```
VIX 越低 → 市场越乐观（越不愿意买保险）→ 指数越高（越危险）。  
阈值：VIX < 15 时，市场进入「过度乐观」区间，历史上往往是大跌前的最后一段平静期。

---

**SPY Put/Call 比率**（实时）
```
P/C 比率 = SPY近期期权 Put 总成交量 / Call 总成交量
```
- < 0.7：极度乐观，几乎无人对冲
- 0.7–1.0：乐观，正常偏多
- 1.0–1.2：中性偏保守
- \> 1.2：恐慌，大量买保险

---

**SOX 偏离 200 日均线**
```
偏离度 = (当前价格 / 200日移动平均) - 1
```
200日均线代表机构认可的「合理估值中枢」。  
偏离 > +30% → 明显过热；偏离 > +50% → 历史罕见的极端状态。

---

### 3.4 复合危机评分（0-100分）

将五个子指标通过 Sigmoid 函数归一化到 [0, 1]，再加权求和：

| 子信号 | 权重 | 含义 |
|--------|------|------|
| SOX 偏离200MA | 25% | 半导体价格是否严重高估 |
| VIX 低位指数 | 25% | 市场情绪是否过度乐观 |
| SOX 6M超额收益 | 20% | 半导体是否异常领涨 |
| 波动率体制比率 | 15% | 波动率是否已经开始扩张 |
| SOX/SPX比率Z值 | 15% | 比率偏离历史的统计显著性 |

**评分解读**

| 分值 | 等级 | 含义 |
|------|------|------|
| 0–40 | 🟢 低风险 | 市场结构健康，无泡沫信号 |
| 40–60 | 🔵 观察期 | 部分指标偏热，持续跟踪 |
| 60–75 | 🟡 警戒 | 多项信号同时偏高，类似历史危机酝酿期 |
| 75–100 | 🔴 高危 | 结构特征与历史顶部高度吻合 |

历史校准参考：科网泡沫顶部（2000-03）评分达 **88.7**；科网酝酿期（1999年）平均 **63.8**。

## 四、图表解读指南

### 图1：`01_sox_spx_history.png` — 长期走势总览

**看什么：** 三个子图从上到下分别是价格、相对强度、恐慌指数。

**红色阴影区** = 危机期间；**橙色阴影区** = 泡沫酝酿期。

- **上图（对数坐标价格）**：两条线的「剪刀差」越大，说明半导体相对大盘越强。2000年顶部前，SOX（蓝线）斜率明显陡于SPX（绿线）。
- **中图（SOX/SPX 比率）**：该比率在2000年初达到历史峰值后崩塌，是最直观的泡沫信号。当前比率若接近或超过历史峰值，需高度警惕。
- **下图（VIX）**：注意危机爆发时VIX会剧烈拉升，但危机**前**往往处于低位——低VIX本身不是危险信号，极低VIX叠加高估值才危险。

---

### 图2：`02_volatility_regimes.png` — 波动率体制

**看什么：** 判断市场是否从平静转入动荡。

- **上图（SOX vs SPX 实现波动率）**：危机期间两者均剧烈上升，但SOX波动率往往更大（高Beta特性）。当SOX波动率突然快速上升而SPX还未反应时，是提前预警信号。
- **下图（体制比率）**：正常值 ≈ 1.0；超过虚线（1.5×）意味着近期波动率已显著超越中期基准，说明危机已在扩散中。该指标**滞后**——它告诉你危机已经开始，而不是即将开始。

---

### 图3：`03_sentiment_indicators.png` — 情绪与领先指标

**这是最关键的一张图**，包含三个独立的领先信号。

- **上图（SOX 6M超额收益）**：蓝色区域持续填充且高于橙色虚线（+20%）时，说明半导体泡沫正在形成。2000年科网泡沫期间，这条线持续超过+40%长达一年多。关键转折点：该指标**从正转负**（蓝色区域消失），往往早于大盘下跌3-6个月出现。
- **中图（VIX + VIX低位指数）**：左轴看VIX本身；右轴（橙色虚线）是2年滚动低位指数，越高意味着当前VIX处于历史越低的分位数，即市场越乐观。两线均处于极值时，是组合信号中最强的一个。
- **下图（均线偏离度）**：SOX的蓝线超过30%（橙色虚线）且与SPX绿线差距持续扩大时，说明半导体估值已明显脱离基本面。

**怎么看：**

**怎么看：**

## 五、当前研究结论（截至 2026-05-08）

### 关键数据

| 指标 | 当前值 | 信号 |
|------|--------|------|
| 复合危机评分 | **75.0 / 100** | 🔴 高危 |
| SOX 偏离200日均线 | **+60.3%** | 历史极端高位 |
| SOX 6个月超额收益 | **+54.9%** | 极端领涨 |
| VIX | **17.19** | 正常偏低，非极端 |
| SPY Put/Call 比率 | **1.199** | 偏保守（有对冲意愿） |
| 波动率体制比率 | **0.91×** | 正常，无扩张 |

### 结论

**当前市场结构与2000年科网泡沫酝酿晚期（1999年底至2000年初）高度相似**，主要体现在：

1. SOX 大幅偏离长期均值，由AI算力需求驱动的半导体泡沫特征明显（NVIDIA带动）
2. SOX 相对 SPX 的超额收益处于历史极端水平，与2000年互联网泡沫中半导体异常领涨的结构完全一致
3. 综合评分75分，已进入历史性顶部区域

**与2000年的重要差异（缓和因素）：**

1. VIX 17（正常），并非2000年顶部前的极端低位（<15），说明市场并未完全失去警惕心
2. Put/Call 比率 1.2，偏向保守，机构存在系统性对冲，与2000年的无保护乐观不同
3. 波动率体制正常，危机尚未进入「扩散阶段」

**需要监控的拐点信号（按重要性排序）：**

1. **SOX 相对 SPX 开始连续跑输**（6M超额收益从正转负）→ 最重要的领先信号
2. **VIX 快速突破 20–25**，且伴随 SOX 回调 → 确认信号
3. **SOX 从近期高点回调超过 15–20%** → 技术性确认
4. **Put/Call 比率下降至 0.7 以下**（乐观情绪极致化）→ 反向指标警报

> **免责声明**：本研究为纯定量分析工具，不构成投资建议。模型基于历史规律构建，对2020年COVID等外生冲击的预警能力有限。市场结构相似不代表结果相同。
